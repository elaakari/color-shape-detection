# 🎨 Real-Time Color & Shape Detection

A simple Computer Vision project built with **Python** and **OpenCV** that detects colored geometric objects in real time using a webcam.

The application identifies the **color** and **shape** of each detected object, draws a bounding box around it, calculates its center coordinates, estimates its area, and displays live statistics such as FPS and the number of detected objects.

## Features

- 🎨 Multi-color detection
- 🔷 Shape recognition
- 📦 Bounding box around detected objects
- 📍 Object center detection
- 📐 Area calculation
- 📊 Live dashboard
- ⚡ Real-time performance (FPS)

## Technologies

- Python
- OpenCV
- NumPy

 

This project was developed as a practical exercise to strengthen Computer Vision fundamentals using OpenCV and image processing techniques.
